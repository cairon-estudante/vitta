## Histórias de Usuário

### Perfil: Nutricionista

**Story N01**  
Como nutricionista, quero visualizar todas as solicitações de consulta pendentes, para analisar os pedidos dos pacientes.

**Story N02**  
Como nutricionista, quero aceitar uma solicitação de consulta, para confirmar o atendimento em um horário específico.

**Story N03**  
Como nutricionista, quero recusar uma solicitação de consulta, para organizar minha agenda conforme minha disponibilidade real.

**Story N04**  
Como nutricionista, quero visualizar minha agenda de consultas confirmadas, para planejar meu dia/semana de atendimentos.

**Story N05**  
Como nutricionista, quero adicionar minhas consultas ao calendário do meu celular, para ter uma visão integrada da minha agenda profissional e pessoal.

**Story N06**  
Como nutricionista, quero cancelar uma consulta já aceita, para ajustar minha agenda em casos de imprevistos.

**Story N07**
Como nutricionista, quero aceitar uma consulta já cancelada, para ajustas minha agenda em caso de mudanças.
